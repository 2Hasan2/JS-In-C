# JavaScript String Methods

JavaScript provides a wide range of built-in methods to work with strings. These methods enable you to manipulate and perform various operations on strings. In this README, we'll cover some commonly used string methods in JavaScript.

## String Properties

- `length`: Returns the length (number of characters) of a string.

## String Manipulation

- `concat()`: Combines one or more strings and returns a new string.

- `toUpperCase()`: Returns a new string with all characters in uppercase.

- `toLowerCase()`: Returns a new string with all characters in lowercase.

- `trim()`: Removes whitespace (spaces, tabs, or line breaks) from the beginning and end of a string.

- `substring()`: Returns a portion of a string between specified indices.

- `slice()`: Returns a portion of a string between specified indices, allowing negative indices.

- `replace()`: Replaces a specified substring or pattern with another string.

- `split()`: Splits a string into an array of substrings based on a specified delimiter.

- `charAt()`: Returns the character at a specified index in a string.

- `charCodeAt()`: Returns the Unicode value (code point) of the character at a specified index.

## Searching and Testing

- `indexOf()`: Returns the index of the first occurrence of a specified substring or character in a string, or -1 if not found.

- `lastIndexOf()`: Returns the index of the last occurrence of a specified substring or character in a string, or -1 if not found.

- `includes()`: Checks whether a string contains a specified substring and returns a boolean.

- `startsWith()`: Checks whether a string starts with a specified substring and returns a boolean.

- `endsWith()`: Checks whether a string ends with a specified substring and returns a boolean.

- `match()`: Searches a string for a specified pattern and returns an array of matched substrings.

- `search()`: Searches a string for a specified pattern and returns the index of the first match.

## String Conversion

- `toString()`: Converts a value to a string.

- `valueOf()`: Returns the primitive value of a string.

This is not an exhaustive list of JavaScript string methods, but it covers some of the most commonly used ones. These methods can be extremely helpful for working with text and manipulating strings in your JavaScript projects.
